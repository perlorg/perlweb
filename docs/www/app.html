Perl has a large number of modules ([% perl_stats.cpan_dists %]+) designed to help you in developing your application(s), these are hosted on CPAN. There are also many other Perl related projects which have their own sites, here are a few of them.

If you are looking for a list of recommended modules, look at Task::Kensho.

CGI/Web Programming and Administration

  • Basics with LWP - an article by Sean Burke
  • CGI.pm - a core module for web forms
  • Development with mod_perl and Apache - persistent, embedded Perl interpreter
  • NMS - Web scripts written by experts
  • embperl - a framework for building websites with Perl.
  • Catalyst - MVC Web Framework similar to Ruby on Rails and Spring
  • CGI::Application - Another MVC Web Framework
  • Template Toolkit - General Purpose Templating System

Science Related Programming

  • BioPerl for Bioinformatics, Genomics, and the Life Science
  • The Perl Data Language for Scientific Computing
  • perl4lib for library and information science

Database Programming

  • DBI.pm - the Perl Database Interface
  • DBIx::Class - Database ORM

Object Oriented Programming

  • Moose - a postmodern object system for Perl 5
  • MooseX:: modules are Moose eXtensions

General Programming

  • The Perl DateTime Project
  • The Perl Email Project
  • POE - a cooperative threading framework
  • PAR - Cross-platform packaging and deployment tool; turns Perl programs into stand-alone executables

GUI Toolkits

  • Perl/Tk - Tk bindings for perl
  • PerlQT - Perl bindings for QT
  • gtk2-perl - Perl bindings for GTK+
  • WxPerl - Perl bindings for the WxWidgets cross platform toolkit

Graphics Programming

  • GD.pm - graphic programming library
