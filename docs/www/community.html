Local Communities

Perl Mongers - find your local Perl group from over [% perl_stats.monger_groups %] groups world wide

Events

Events - Perl conferences, hackathons and workshops and local groups around the world

Mail Lists

lists.perl.org has information on all the Perl mailing lists you can join. If you're just starting, you probably want to join perl-beginners.

Community Sites

  • Perl.org - You are here
  • blogs.perl.org - Personal Perl blogs
  • PerlMonks - A community committed to sharing Perl knowledge and coding tips. The site contains questions and answers, useful snippets, and a library of code
  • Planet Perl Iron Man - open Perl blogging challenge
  • Perlsphere - another aggregation of selected Perl blogs

Chat

IRC (Internet Relay Chat) enables communities to gather in 'channels' on-line and chat in real-time. It's a great place to get help if you're polite and ask smart questions.

  • irc.perl.org (channel list)
  • Freeenode's #perl channel on Scrollback

Organizations

  • The Perl Foundation - dedicated to the advancement of the Perl programming language through open discussion, collaboration, design, and code
  • Enlightened Perl Organization - a membership organization created to support the new wave of CPAN (Catalyst, DBIx::Class, Moose, etc.) and to evangelize the new Perl enlightenment
  • YAPC Europe Foundation - The YAPC::Europe Foundation (YEF) is a non-profit organization which works to promote Perl in Europe primarily through public events
