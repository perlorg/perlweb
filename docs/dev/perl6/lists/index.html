Current mailinglists

To subscribe to a list, send mail to listname-subscribe@perl.org. The mailinglist manager has a few nifty features. Mail listname-help@perl.org to get information about them. The most popular feature is the "get" function you can use to request old mails from the archive. Very useful if you join a list and want to catch up on what happened before you subscribed.

You might find the FAQ useful.

You can also browse the List of all Perl mailing lists.

perl6-announce
Moderated list for news of new lists, working groups, and so on. Summaries from the top-level working groups are also posted here.
Subscribe: perl6-announce-subscribe@perl.org - Archive.

perl6-language
Discusson of the perl6 language.
Subscribe: perl6-language-subscribe@perl.org - Archive.

perl6-internals
Discusson of parrot development
Subscribe: perl6-internals-subscribe@perl.org - Archive.

perl6-compiler
Discusson of developmen of the perl6 compiler
Subscribe: perl6-compiler-subscribe@perl.org - Archive.

perl6-all
This is a "metalist" that gets all mails from all of the above mentioned lists. If you subscribe to perl6-all you'll always get all mails to all perl 6 mailinglists. You can setup procmail to filter on the X-Mailing-List header to filter on the original listname. You can't post directly to the perl6-all list, only through the "real lists". Subscribe by sending mail to perl6-all-subscribe@perl.org.

There are also many mailing lists that were used early in the perl6 process that are no longer being used.

Please don't include the posting addresses in links on the web. Always use either the subscribe address or a link to an archive.

List archives

These lists are all archived at www.nntp.perl.org. The archive is not searchable yet, but it will be soon.

Most @perl.org mailinglists are also archived at the mail archive. The url for each list is http://www.mail-archive.com/listname@perl.org
