Reporting Issues

Before reporting an issue check log.perl.org and the perl.org twitter feed to ensure that there are no known issues/outages.

To report content issues with {www,learn,dev,lists}.perl.org, please use our issue tracker on GitHub. You can also clone the git repository and send us a pull request.

For all other issues with perl.org services, please send an email to webmaster at perl.org.
